# Photo Organizer 📸

로컬 사진 폴더를 **연도/월별로 자동 정리**하는 웹 애플리케이션입니다.

## 기능

- 📅 **EXIF 메타데이터** 기반 정렬 (사진 촬영 날짜)
- 📝 **수정일** 자동 감지 (EXIF 없을 때)
- 👁️ **미리보기** - 변경 전 파일 목록 확인
- 🔄 **진행 상황 표시** - 실시간 진행률 추적
- 🛡️ **안전한 정렬** - 중복 파일명 자동 처리 & 빈 폴더 정리
- 🌙 **다크테마 UI** - 편안한 인터페이스

## 설치 & 실행

### 1️⃣ Node.js 설치
[nodejs.org](https://nodejs.org/) 에서 LTS 버전 설치 (v18 이상 권장)

### 2️⃣ 프로젝트 설정
```bash
# 프로젝트 폴더로 이동
cd photo-organizer

# 패키지 설치
npm install
```

### 3️⃣ 서버 시작

**Windows (가장 간단):**
```bash
run.bat
```

**macOS / Linux (터미널 사용):**
1. 터미널(Terminal) 앱을 엽니다.
2. 프로젝트 폴더로 이동한 후 아래 명령어를 입력합니다:
```bash
# 실행 권한 부여 (처음 한 번만 필요)
chmod +x run.sh

# 앱 실행
./run.sh
```

**일반 명령어:**
```bash
npm start
```

### 4️⃣ 브라우저에서 접속
```
http://localhost:3000
```

## 사용 방법

1. **폴더 경로 입력**
   - 정렬할 사진 폴더 경로 입력
   - 예: `D:\Photos` 또는 `/Users/name/Photos`

2. **스캔 버튼 클릭**
   - 폴더 분석 시작
   - 생성될 폴더 구조 미리 확인

3. **파일 목록 확인**
   - 현재 위치 → 이동 위치
   - 날짜 소스 (EXIF vs 수정일)
   - 이미 정렬된 파일은 자동 제외

4. **정렬 실행**
   - `정렬 실행` 버튼 클릭
   - 파일 이동 진행 상황 확인
   - 완료 후 결과 요약

## 폴더 구조 예

정렬 **전:**
```
📁 Photos/
  ├─ IMG_001.jpg
  ├─ photo.png
  ├─ vacation_pic.heic
  └─ various_files...
```

정렬 **후:**
```
📁 Photos/
  ├─ 2024/
  │  ├─ 01/
  │  │  ├─ IMG_001.jpg
  │  │  └─ photo.png
  │  └─ 06/
  │     └─ vacation_pic.heic
  └─ 2023/
     └─ 12/
        └─ various_files...
```

## ⚠️ 주의사항

- **백업 먼저!** 중요한 파일은 정렬 전 백업하세요
- **권한 확인** - 폴더에 읽기/쓰기 권한이 필요합니다
- **중복 파일** - 같은 이름 파일이 있으면 `_1`, `_2` 식으로 자동 변경됩니다

## 지원 형식

**이미지:**
- JPG, JPEG, PNG, GIF, WebP, HEIC, TIFF, BMP

**동영상:**
- MP4, MOV, AVI, MKV

**기타:**
- RAW, PSD, PDF 등 모든 파일 지원

## 문제 해결

### 포트 3000이 이미 사용 중인 경우
```bash
# Windows
netstat -ano | findstr :3000
taskkill /PID [PID] /F

# macOS/Linux
lsof -i :3000
kill -9 [PID]
```

### EXIF 데이터가 없는 경우
자동으로 파일 수정일을 기준으로 정렬됩니다.

### 특정 폴더에 접근 불가인 경우
관리자 권한으로 실행하거나, 접근 가능한 폴더를 선택해주세요.

## 🍎 macOS 사용자 가이드

### 1️⃣ 빌드 방법 (GitHub Actions)
이 앱은 GitHub Actions를 통해 빌드됩니다. (상단 `Build/Release` 워크플로우 참고)
1. 수정된 코드를 GitHub 리포지토리에 푸시합니다.
2. `v1.3.0`과 같은 태그를 생성하고 푸시하면 자동으로 빌드가 시작됩니다:
   ```bash
   git tag v1.3.0
   git push origin v1.3.0
   ```
3. GitHub의 **Releases** 메뉴에서 각 OS에 맞는 파일을 다운로드할 수 있습니다.

### 2️⃣ 실행 시 주의사항 (보안 설정)
애플 개발자 등록이 되어 있지 않아 앱을 처음 실행할 때 "확인되지 않은 개발자" 메시지가 뜰 수 있습니다.
- **해결 방법**: 응용 프로그램 폴더의 앱 아이콘에서 **마우스 오른쪽 버튼 클릭 -> 열기**를 선택하고, 안내창에서 다시 **'열기'**를 클릭하세요.

## 기술 스택

- **Backend:** Node.js + Express
- **Frontend:** Vanilla JavaScript (프레임워크 없음)
- **메타데이터:** exifr (EXIF 추출)
- **CORS:** 로컬 개발 지원
